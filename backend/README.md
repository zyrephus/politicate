## UV Commands

### Adding a Package

```sh
uv add <package_name>
```

### Installing
```sh
uv install
```

### Running a file
```sh
uv run <name_of_file>
```

### FastAPI
```sh
uv run fastapi dev
```
Link to docs -> https://docs.astral.sh/uv/guides/integration/fastapi/#migrating-an-existing-fastapi-project
